# Chat App

A chat app server / client implementation for DECS course project.

## Features
- Login
- Channel Messages
- Private Messages
- File upload
- Persistent messaging

## Internals
- Thread Pool
- Database Pool


## Usage
**Server**
```sh
cd src/server/
make
./server <port>
```

**Client**
```sh
cd src/client/
make
./client <ip> <port>
```
